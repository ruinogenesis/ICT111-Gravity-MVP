# User Consent Statement — Gravity

This prototype is created for ICT111 learning purposes at Rangsit University. We will ask users to test the Gravity prototype (RSU student secondhand marketplace) and provide feedback. The information collected will be used only to improve the project prototype. Users should not provide sensitive personal information. Participation is voluntary and can stop at any time.

## Data Collected
- Name/nickname or anonymous ID: anonymous tester code only (e.g., TU-01)
- Feedback score: 1–5 ratings per task
- Comments about usability: anonymized notes (no names, no exact addresses)
- Task completion result: yes/no and time per task

## Data Use
The data will be used for class project analysis, prototype improvement, and final project reporting in the group GitHub repository (public). Because the repository is public, only anonymized codes and non-identifying comments are ever recorded.

## Consent Record
- Lab 03 problem interviews (R001–R015): verbal consent using the consent questions in `/docs/customer-questions.md`; anonymized notes only.
- Lab 08 validation testing (TU-01–05): verbal consent before each session; results stored under anonymous codes in `/data/validation-results.csv`.
- Lab 13 user testing (planned): this written statement will be shown to each tester before the session begins.
