# Experiment Script — Gravity

## 1. Opening Script
"Thank you for helping us test our prototype. This is not a test of you — we are testing whether our marketplace idea and interface are clear. Please think aloud: say what you're looking for, what you expect to happen, and anything that confuses you. There are no wrong answers, and honest criticism helps us most."

## 2. Test Scenario
"Imagine the semester is ending. You are moving out and need to sell a desk and some textbooks within a few days. Other students are moving in and looking for cheap used items. You'll use Gravity, a secondhand marketplace for RSU students only."

Demo link: see /prototype/mvp-demo-link.md. Tester uses their own phone if possible (tests FR-14).

## 3. Tasks for Tester
| Task No. | Task Instruction | Related Requirement | Expected Result |
|---|---|---|---|
| T1 | Open the homepage. In your own words, what is this website for, and who can use it? | FR-01 | Tester states: buy/sell used items, RSU students only |
| T2 | Sell your study desk (800 baht, good condition, pickup Dorm A). Post it as a listing. | FR-03, FR-10, FR-11 | Tester completes the form, sees the success confirmation, finds their desk in the list |
| T3 | You want a cheap textbook. Find one that costs less than 300 baht. | FR-05, FR-06 | Tester uses search or the Textbooks filter and identifies GL-003 (250 THB) |
| T4 | Open that textbook's detail page. Would you trust this seller? What tells you that? | FR-07, FR-15 | Tester opens the detail view and mentions the "RSU verified" badge |
| T5 | Which items can you still get, and which are gone? What does "Reserved" mean? | FR-08 | Tester correctly interprets Available / Reserved / Sold |
| T6 | Press "Request contact". What happens next — and why can't you see the seller's LINE yet? | FR-02, FR-15 | Tester explains: seller must accept first, then contact is revealed to both |

## 4. Observation Notes
For each task record in /data/experiment-results.csv: completed (Yes/No), time in seconds, help needed (Yes/No), confusion point or wrong click, clarity 1–5, improvement note. Do not help unless fully stuck — if you do, mark help needed = Yes.

## 5. Closing Questions (neutral)
- What was easy to understand?
- What was confusing or unclear?
- Which feature would matter most to you personally, if any?
- What is missing that you would need before using this for real?
- Next time you need to sell something — would you post here or in a Facebook/LINE group? Why?
