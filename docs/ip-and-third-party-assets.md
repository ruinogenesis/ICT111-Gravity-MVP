## IP Summary
Gravity currently uses **no third-party icons, images, fonts files, UI kits, CSS/JS libraries, APIs, or external datasets** — every line of code, all sample data, wireframes, and diagrams are team-created, which was a deliberate feasibility decision (technical-architecture.md). The only external elements are the lecturer's fixed requirement framework (attributed), standard tooling (GitHub, Mermaid, system fonts), and disclosed AI assistance for documentation, with the team reviewing and taking responsibility for all content. Rule going forward: **no asset enters the prototype before it enters this register** with source and license — this especially applies to any icons or placeholder images added during the Lab 10–11 photo feature and the Lab 12 landing page.
